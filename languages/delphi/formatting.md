# Delphi Formatting & Style

## Standards

*   **Indentation**: 2 spaces. No tabs.
*   **Margins**: 120 characters.

## Code Organization

*   **Class Layout**:
    1.  `private` fields
    2.  `private` methods
    3.  `protected`
    4.  `public` properties
    5.  `public` constructors/destructors
    6.  `public` methods

## Comments

*   Use XML Documentation `///` for all public methods to enable hovering support.
